# Sparql -- basics

	SELECT ?a ?b ?c
	WHERE
	{
	  x y ?a.
	  m n ?b.
	  ?b f ?c.
	}
	LIMIT 10

Note that `x y ?a.`

..means "x relates-in-way-y to value ?a"

..e.g. `y` could be... "hasColor", "car hasColor green" ... or some other arbitrary relationship indicator.

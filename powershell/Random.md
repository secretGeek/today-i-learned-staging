# Get a random number in Powershell

Get a random number with Get-Random

## Random number from 0 to 99

	Get-Random 100


## Random Number from 0.0 to 1.0

	Get-Random 1.0


## Random Number from a set of 3 numbers

	Get-Random 12,24,36
